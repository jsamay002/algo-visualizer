# Algorithm Visualizer

An interactive web app that visualizes common competitive programming algorithms step by step.

## How to Use
1. Open `index.html` in your browser.
2. Select an algorithm from the dropdown.
3. Enter an array (comma separated).
4. Click **Start** and **Step** to visualize the algorithm's execution.

## Features
- Binary Search visualization
- Prefix Sums visualization
- Two Pointers visualization
- Step-by-step execution

## Tech Stack
- HTML
- CSS
- JavaScript

## Motivation
Built to help understand competitive programming techniques more intuitively.
