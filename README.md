# Algorithm Visualizer

An interactive web app that visualizes common competitive programming algorithms step by step.

---

## How to Use
1. Open `index.html` in your browser.
2. Select an algorithm from the dropdown.
3. Enter an array (comma separated).
4. Click **Start** and **Step** to visualize the algorithm’s execution.

---

## Features

### Search / Array Techniques
- Binary Search
- Prefix Sums
- Two Pointers

### Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Bogo Sort

### Visualization
- Step-by-step execution with visual feedback
- Histogram-style bar charts for arrays
- Highlights active indices and pointers
- Displays pseudocode and time complexity for each algorithm

---

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js (for visualizations)

---

## Motivation
Built to help understand competitive programming techniques more intuitively by providing a visual and interactive learning experience.

---

## Time Complexity Reference

| Algorithm        | Best Case     | Average Case  | Worst Case    |
|------------------|---------------|---------------|---------------|
| Binary Search    | O(1)          | O(log n)      | O(log n)      |
| Bubble Sort      | O(n)          | O(n²)         | O(n²)         |
| Selection Sort   | O(n²)         | O(n²)         | O(n²)         |
| Insertion Sort   | O(n)          | O(n²)         | O(n²)         |
| Merge Sort       | O(n log n)    | O(n log n)    | O(n log n)    |
| Quick Sort       | O(n log n)    | O(n log n)    | O(n²)         |
| Heap Sort        | O(n log n)    | O(n log n)    | O(n log n)    |
| Bogo Sort        | O(n)          | O((n+1)!)     | O(∞)          |
